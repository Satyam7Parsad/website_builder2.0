## ğŸ“¸ Background Image Tips

**What to look for:**
- Dark coding screen (HTML/CSS/JavaScript)
- Blue/purple tones
- Code clearly visible but not distracting
- High resolution (1920x1080+)

**Where to find:**
- Your own screenshots of VS Code/IDE
- Unsplash.com (search: "coding screen dark")
- Generate with AI image tools

**In the designer:**
1. Select Hero section
2. Scroll to "BACKGROUND IMAGE"
3. Click "Upload Background Image"
4. Select your coding screen image
5. Adjust "Overlay Opacity" to **0.80**
